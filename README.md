
# Afnan Predictor (Magic Chess GOGO)

A simple Streamlit web app to predict next opponents in Magic Chess GOGO using an adaptive loop model.
Created by Aby.

## Files
- `afnan_predictor.py` - Streamlit app
- `requirements.txt` - Python dependencies

## How to deploy to Streamlit Cloud (quick)
1. Create a GitHub repository (e.g., `afnan-predictor`) and upload all files from this package.
2. Go to https://share.streamlit.io/ and log in.
3. Click **New app** -> select the GitHub repo, branch `main`, and the file `afnan_predictor.py` -> Deploy.
4. Open the provided link on your Android phone.

## How to run locally (PC)
1. Install Python (3.8+).
2. `pip install -r requirements.txt`
3. `streamlit run afnan_predictor.py`
4. Open `http://localhost:8501` or `http://<PC-IP>:8501` on your phone.
